# 🏃‍♂️ AI 피트니스 코치 - 청소년 운동 지속 도우미

> **"청소년에게 필요한 건 더 많은 운동 정보가 아니라, 포기하지 않게 만드는 구조다."**

## 🎯 프로젝트 개요

청소년들이 운동을 **시작하지 못해서**가 아니라 **지속하지 못하는** 문제를 해결하는 AI 기반 피트니스 코치입니다.

### 핵심 철학
- ✅ **실패는 정상** - 포기하지 않게 격려
- ✅ **5분도 성공** - 작은 성취도 인정  
- ✅ **비교 없음** - 개인 맞춤 성장
- ✅ **구조적 지원** - 의지가 아닌 시스템으로 해결

## 🚀 주요 기능

### 🤖 실시간 AI 분석
- **자연어 입력**: "운동 못했어 우울해" → AI가 상황 분석
- **ML 기반 예측**: 포기 위험도 실시간 계산
- **패턴 매칭**: 30명의 실제 데이터 기반 유사 사례 찾기

### 💬 개인화된 코칭
- **성격별 맞춤**: 격려형, 도전형, 분석형, 친근형
- **상황별 조언**: 실패/성공/고민 상황에 따른 차별화된 메시지
- **난이도 자동 조절**: 위험도에 따라 운동 강도 추천

## 🛠️ 기술 스택

- **Backend**: FastAPI, Python
- **AI/ML**: scikit-learn, 자연어 처리
- **Database**: JSON 파일 기반 (확장 가능)
- **Interface**: 웹 UI, CLI 채팅, REST API
